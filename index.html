About me

I'm Umakant Yadav, a Full Stack Web Developer and an aspiring Data Analyst from Surat, India. I work primarily with PHP, JavaScript (jQuery), Bootstrap, MySQL and also use Python and Java in various projects. I enjoy turning complex problems into simple, robust and maintainable solutions.

My work includes building dynamic admin panels with DataTables, generating bulk QR codes and ZIP export tools, integrating OpenCV (face detection & crop) with PHP, and building PHP → Python bridges for Indic NLP transliteration. I also experiment with data workflows and analytics to move towards Data Analyst roles.
